# 온라인 회의에서 내 목소리 + 상대방 목소리 함께 전사하기 (macOS)

화상 회의(Zoom, Google Meet, Teams 등)에서 **내 목소리와 상대방 목소리를 모두**
전사하려면, 시스템으로 재생되는 상대방 오디오를 마이크처럼 입력으로 되돌려야 합니다.
macOS는 기본적으로 시스템 출력 오디오를 앱이 직접 캡처하도록 허용하지 않으므로,
**BlackHole**(무료 가상 오디오 장치)과 **집계 장치(Aggregate Device)** 를 함께 사용합니다.

> 이 플러그인은 입력 장치 선택 기능을 제공합니다. 아래 절차로 "마이크 + 시스템 오디오"를
> 하나로 묶은 가상 입력 장치를 만들면, 플러그인 마이크 드롭다운에서 선택하기만 하면 됩니다.

---

## 동작 원리

```
[내 마이크] ───────────────┐
                           ├──▶ [집계 장치] ──▶ 플러그인이 입력으로 캡처 ──▶ AWS Transcribe
[회의 앱 출력] ─▶ [BlackHole] ┘
                  │
                  └─▶ [멀티 출력 장치] ──▶ 실제 스피커/헤드폰 (내가 듣는 소리)
```

- **BlackHole**: 시스템 출력 오디오를 입력으로 되돌리는 가상 케이블.
- **멀티 출력 장치(Multi-Output Device)**: 상대방 소리를 BlackHole로 보내면서
  동시에 내 스피커로도 들리게 함. (없으면 회의 소리를 못 듣게 됨)
- **집계 장치(Aggregate Device)**: 내 마이크 + BlackHole을 하나의 입력 장치로 묶음.

---

## 1단계: BlackHole 설치

Homebrew로 설치하는 것이 가장 간단합니다.

```bash
brew install blackhole-2ch
```

설치 후 Mac을 재시작하거나 로그아웃/로그인하면 오디오 장치 목록에 `BlackHole 2ch`가 나타납니다.

> Homebrew가 없다면 [BlackHole 공식 페이지](https://existential.audio/blackhole/)에서
> 설치 프로그램을 내려받을 수 있습니다(이메일 등록 후 무료 다운로드).

---

## 2단계: 멀티 출력 장치 만들기 (회의 소리를 들으면서 캡처)

상대방 소리를 BlackHole로만 보내면 내 귀에는 안 들립니다. 그래서 스피커와 BlackHole에
**동시에** 출력하는 장치를 만듭니다.

1. `응용 프로그램 ▸ 유틸리티 ▸ Audio MIDI 설정` 실행
2. 왼쪽 아래 `+` 버튼 ▸ **다중 출력 장치 생성(Create Multi-Output Device)**
3. 오른쪽 목록에서 다음 두 개를 체크
   - 실제 출력 장치 (예: `MacBook Pro 스피커` 또는 사용 중인 헤드폰)
   - `BlackHole 2ch`
4. 이름을 알아보기 쉽게 변경 (예: `회의용 출력`)
5. **드리프트 보정(Drift Correction)**: 실제 스피커가 아닌 `BlackHole 2ch` 쪽에 체크하는 것을 권장

> 목록 **맨 위에 있는 장치**가 마스터 클록이 됩니다. 보통 실제 스피커를 맨 위에 둡니다.

---

## 3단계: 집계 장치 만들기 (마이크 + 상대방 소리 합치기)

내 목소리와 상대방 목소리를 한 입력으로 받기 위해 둘을 묶습니다.

1. 같은 `Audio MIDI 설정` 창에서 `+` 버튼 ▸ **집계 장치 생성(Create Aggregate Device)**
2. 오른쪽 목록에서 다음 두 개를 체크
   - 내 마이크 (예: `MacBook Pro 마이크` 또는 외장 마이크)
   - `BlackHole 2ch`
3. 이름 변경 (예: `회의 전사 입력`)
4. **드리프트 보정**: `BlackHole 2ch` 쪽에 체크 권장
5. 목록 맨 위(마스터 클록)에 **내 마이크**를 두는 것을 권장

---

## 4단계: 오디오 라우팅 설정

### 시스템 출력

`시스템 설정 ▸ 사운드 ▸ 출력`에서 **`회의용 출력`(멀티 출력 장치)** 선택.
→ 상대방 목소리가 내 스피커로 들리면서 동시에 BlackHole로도 전달됩니다.

### 회의 앱 설정

Zoom / Meet / Teams 등에서:

- **스피커**: `회의용 출력`(또는 시스템 기본값) — 상대방 소리가 BlackHole로 가야 함
- **마이크**: 회의 상대방에게는 평소처럼 **실제 마이크**를 그대로 사용
  (집계 장치를 회의 앱 마이크로 쓰면 에코가 생길 수 있으므로 권장하지 않음)

### 플러그인 설정

이 플러그인의 마이크 선택 드롭다운에서 **`회의 전사 입력`(집계 장치)** 선택.
→ 내 마이크 + 상대방 소리가 함께 전사됩니다.

---

## 빠른 점검 체크리스트

- [ ] `brew install blackhole-2ch` 완료, 장치 목록에 `BlackHole 2ch` 표시됨
- [ ] 멀티 출력 장치(`회의용 출력`) 생성: 실제 스피커 + BlackHole 체크
- [ ] 집계 장치(`회의 전사 입력`) 생성: 실제 마이크 + BlackHole 체크
- [ ] 시스템 출력 = `회의용 출력`
- [ ] 회의 앱 스피커 = `회의용 출력`, 마이크 = 실제 마이크
- [ ] 플러그인 마이크 = `회의 전사 입력`
- [ ] 테스트: 유튜브 영상을 틀고 직접 말하면서, 두 소리가 모두 전사되는지 확인

## 자주 겪는 문제

| 증상 | 원인 / 해결 |
|------|------------|
| 상대방 소리가 전사 안 됨 | 시스템/회의 앱 출력이 BlackHole(또는 멀티 출력)로 가지 않음. 출력 장치 재확인 |
| 회의 소리가 안 들림 | 출력을 BlackHole 단독으로 설정함 → 반드시 **멀티 출력 장치**를 사용 |
| 내 목소리만 전사됨 | 플러그인 입력이 집계 장치가 아니라 일반 마이크로 설정됨 |
| 소리가 끊기거나 찢어짐 | 드리프트 보정을 BlackHole 쪽에 체크. 집계/멀티 출력의 장치 순서(마스터 클록) 조정 |
| 에코 발생 | 회의 앱 마이크로 집계 장치를 쓰지 말 것. 회의 앱에는 실제 마이크만 사용 |
| 한쪽 소리가 작음 | 마이크와 BlackHole의 입력 게인 차이. `Audio MIDI 설정`에서 채널별 볼륨 조정 |

## 더 간단한 대안 (유료)

장치 구성이 번거롭다면 [Loopback](https://rogueamoeba.com/loopback/)(Rogue Amoeba, 유료)으로
"마이크 + 시스템 오디오"를 단일 가상 입력으로 한 번에 믹스할 수 있습니다. 설정 후
플러그인에서 그 가상 장치를 선택하면 됩니다.

## 참고: 녹음/전사 시 고지

상대방의 음성을 전사·기록하는 것은 지역과 상황에 따라 **사전 동의가 필요**할 수 있습니다.
회의 참가자에게 전사/기록 사실을 안내하고 동의를 받는 것을 권장합니다.
